== Description ==

This plugin allows you to easily create a CrowdFunding portal/website by using Thrinacia Atlas infrastructure. You can setup your CrowdFunding Website powered by Thrinacia Atlas under chosen URL path for the full CrowdFunding portal experience. For more information about Thrinacia Atlas please visit this page https://www.thrinacia.com/atlas

If you experience any problems during the setup of this plugin then please visit https://www.thrinacia.com/blog/post/setting-up-the-thrinacia-atlas-wordpress-plugin for troubleshooting.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Setup your Thrinacia Atlas path by going to Settings and then Thrinacia Atlas
